# Classificador Financeiro com IA Local (Ollama)

Descrição: Script Python que utiliza um LLM local (via Ollama/Qwen) para classificar automaticamente linhas de despesas financeiras cruzando "Conta Contábil" com "Centro de Custo".

Destaques:

Usa Engenharia de Prompt (Few-Shot) para ensinar regras de controladoria para a IA.

Roda 100% local (Privacidade de dados financeiros).

Processou 2500 linhas que seriam classificadas manualmente.

Stack: Python, Requests, Ollama (Qwen 2.5).
